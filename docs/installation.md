# Installation

## Stable release

To install CCIAtlasProj, run this command in your terminal:

```sh
uv add CCIAtlasProj
```

Or if you prefer to use `pip`:

```sh
pip install CCIAtlasProj
```

## From source

The source files for CCIAtlasProj can be downloaded from the [Github repo](https://github.com/xfolka/CCIAtlasProj).

You can either clone the public repository:

```sh
git clone git://github.com/xfolka/CCIAtlasProj
```

Or download the [tarball](https://github.com/xfolka/CCIAtlasProj/tarball/master):

```sh
curl -OJL https://github.com/xfolka/CCIAtlasProj/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd CCIAtlasProj
uv pip install .
```
